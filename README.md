Persona

O aplicativo foi projetado principalmente para estudantes universitários e jovens adultos que dividem apartamento, república ou despesas de viagens com amigos.

O foco foi facilitar a organização financeira coletiva de forma simples, evitando discutir sobre contas esquecidas, valores errados ou pagamentos atrasados.

Heurística de Nielsen em foco Visibilidade do Status do Sistema

A heurística priorizada foi a visibilidade do status do sistema.

O protótipo foi desenvolvido para deixar sempre claro:

quem deve quanto deve para quem deve e o status do pagamento

Isso reduz a insegurança e a ansiedade ao lidar com o dinheiro, transmitindo maior transparência para os usuários.

Fluxo de Tarefa Cadastro de uma conta de luz dividida entre 3 pessoas O usuário acessa o Dashboard de grupos Selecionado o grupo da casa/apartamento Clique em “Adicionar Gasto” Preenchimento: descrição da conta valor total participantes da divisão Confirmação do cadastro O sistema calcula automaticamente quanto cada pessoa deve O saldo é atualizado na tela de pagar Considerações Finais

O protótipo foi desenvolvido em baixa fidelidade com foco na clareza das informações financeiras e facilidade de navegação.

As telas foram organizadas para reduzir a confusão durante tarefas repetitivas, principalmente em situações envolvendo cobrança e divisão de gastos.
